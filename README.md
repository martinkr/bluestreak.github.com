# bluestreak-docs

Static Vite site for Bluestreak marketing + documentation.

## Pages

- `/` - balanced, engaging marketing page with benefits and use cases
- `/docs/` - practical integration docs with code examples for:
  - Pino
  - Winston
  - Browser `console.log`
  - Node stream pipelines

## Commands

```bash
npm install
npm run dev
npm run build
npm run preview
```
